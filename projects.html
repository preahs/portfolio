Home Lab — Since 2024

Proxmox Homelab
Dell PowerEdge R730

A bare-metal Proxmox VE 8 hypervisor running 25+ LXC containers. Self-hosted infrastructure covering media, productivity, finance, network security, and monitoring — all behind Cloudflare tunnels and a reverse proxy layer.

  • Proxmox VE 8 on Dell PowerEdge R730 bare-metal
  • 25+ LXC containers — Jellyfin, Nextcloud, Vaultwarden, Grafana, AdGuard Home, Forgejo, and more
  • Cloudflare Tunnels + Nginx Proxy Manager for external access with zero open inbound ports
  • VLAN segmentation via pfSense for network isolation between trusted/untrusted traffic
  • Automated backups with Proxmox Backup Server to a dedicated storage node
  • Grafana + Prometheus + node_exporter dashboards for real-time host and container metrics
  • Full documentation written throughout — setup guides, troubleshooting logs, architecture diagrams
